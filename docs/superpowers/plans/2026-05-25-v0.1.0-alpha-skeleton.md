# v0.1.0-alpha — Monorepo Skeleton Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** `git clone && pnpm run dx` spins up a working development environment in under 5 minutes.

**Architecture:** pnpm workspace monorepo with Turbo orchestration. Four packages (`web`, `worker`, `packages/shared`, `ee`) each compile and export. Docker Compose runs Postgres, ClickHouse, Redis, MinIO. Toolchain (ESLint, Prettier, Husky) is configured and enforced.

**Tech Stack:** Node.js 26, pnpm 11.1.3, Turbo 2.x, TypeScript 5.8, Next.js 15 (Pages Router), Express, BullMQ, Prisma, ClickHouse, Tailwind CSS, shadcn/ui.

---
