# v0.8.0-beta ‚Äî Annotation Queues Implementation Plan

**Goal:** Build team-based manual annotation workflow: queue CRUD, item assignment, annotator workbench, review flow.

**Architecture:** Pure web-side implementation (no worker). Annotation results write to existing `Score` model with `source=ANNOTATION`. Queue status tracked in `AnnotationQueueItem`.

### Task 1: Add AnnotationQueue and AnnotationQueueItem to Prisma schema

**Files:**

- Modify: `packages/shared/prisma/schema.prisma`

- [ ] **Step 1: Append models, enums, and relations**

Add after existing `EvalRun` model:

```prisma
// ‚îÄ‚îÄ‚îÄ Annotation Queues ‚îÄ‚îÄ‚îÄ
enum AnnotationStatus {
  PENDING
  ASSIGNED
  ANNOTATED
  REVIEWED
}

model AnnotationQueue {
  id            String   @id @default(cuid())
  createdAt     DateTime @default(now()) @map("created_at")
  updatedAt     DateTime @default(now()) @updatedAt @map("updated_at")
  projectId     String   @map("project_id")
  project       Project  @relation(fields: [projectId], references: [id], onDelete: Cascade)

  name          String
  description   String?
  scoreConfigId String?  @map("score_config_id")
  scoreConfig   ScoreConfig? @relation(fields: [scoreConfigId], references: [id], onDelete: SetNull)

  items         AnnotationQueueItem[]

  @@unique([projectId, name])
  @@index([projectId])
  @@map("annotation_queues")
}

model AnnotationQueueItem {
  id            String   @id @default(cuid())
  createdAt     DateTime @default(now()) @map("created_at")
  updatedAt     DateTime @default(now()) @updatedAt @map("updated_at")
  queueId       String   @map("queue_id")
  queue         AnnotationQueue @relation(fields: [queueId], references: [id], onDelete: Cascade)
  traceId       String   @map("trace_id")

  status        AnnotationStatus @default(PENDING)
  assignedTo    String?  @map("assigned_to")
  annotator     User?    @relation("Annotator", fields: [assignedTo], references: [id], onDelete: SetNull)

  scoreValue    Float?   @map("score_value")
  stringValue   String?  @map("string_value")
  comment       String?

  reviewedBy    String?  @map("reviewed_by")
  reviewer      User?    @relation("Reviewer", fields: [reviewedBy], references: [id], onDelete: SetNull)

  @@index([queueId, status])
  @@index([assignedTo, status])
  @@map("annotation_queue_items")
}
```

Add to `Project` model:

```prisma
  evalTemplates     EvalTemplate[]
  evalRuns          EvalRun[]
  annotationQueues  AnnotationQueue[]
```

Add to `User` model:

```prisma
  assignedItems     AnnotationQueueItem[] @relation("Annotator")
  reviewedItems     AnnotationQueueItem[] @relation("Reviewer")
```

Add to `ScoreConfig` model:

